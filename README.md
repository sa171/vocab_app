# vocab_app

Purpose:
  Creation of a vocab app that will fetch words and their meanings which are presented before user as a  questionare.
  
 Functionality:
 1. Authentication(basic, token)
 2. Login (basic , Oauth)
 3. Flashcards
 4. Save progress
 

 API: https://api.dictionaryapi.dev/api/v2/entries/en_US
